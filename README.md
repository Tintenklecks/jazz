# Dusita Telefonhilfe

Eine mobile Progressive Web App, die mit Thai-Übersetzungen Schritt für Schritt durch deutschsprachige Telefonate im Dusita Thai Spa führt.

Die drei farbcodierten Gesprächswege decken das Buchen, Absagen und Verschieben eines Termins ab. Eine jederzeit erreichbare Schnellhilfe enthält Verständnishilfen, Antworten auf häufige Fragen, Gutschein-Sätze und eine Buchstabierhilfe.

Zum lokalen Starten im Projektordner:

```sh
python3 -m http.server 8080 -d assistant
```

Danach `http://localhost:8080` im Browser öffnen. Nach dem ersten Laden funktioniert die installierte App auch offline.
